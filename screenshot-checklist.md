## Naming Convention
- pricing.png — pricing page screenshot
- interface.png — main UI/dashboard screenshot  
- feature.png — key differentiating feature screenshot

## How to take screenshots
1. Visit the tool's website
2. Capture at 1280px wide (standard desktop)
3. Save as PNG in the correct folder
4. Keep file size under 200KB (compress if needed)
5. Once all screenshots for a tool are saved, run the insert prompt
